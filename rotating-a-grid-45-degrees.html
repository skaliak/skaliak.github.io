    Rotating a grid 45 degrees

    Some old code revisited

    About 9 years ago, my primary programming language was Perl. I had not been programming that long, and everything I knew was self-taught. A few years prior to that I had bought a book about C++ (I think it was "teach youself c++ in 21 days" or something like that.) But I did not make much progress with it. I think I got frustrated and quit when pointers were introduced. What finally forced me to learn how to write a functional program was necessity -- I needed to automate some task at work, and I knew I could do it with Perl.

    Anyway, in 2006, I started to work on writing a "word search" solver. I figured that the easiest approach would be to search left to right, row by row, then rotate the whole grid of letters, and repeat. This would also handle searching backwards, since when the grid was rotated 180 degrees, the letters would have reversed. Writing a function to rotate 90° was easy, but I remember struggling with the 45° rotation for a while.

    Eventually I got something working:

vector<string> Rotate45(vector<string> in)
{
    vector<string> rotated;
    int lastind = in[0].length() - 1;
    for (int i = lastind; i >=0; i--)
    {
        vector<char> charv;
        int o = i;
        for (int x = 0; x <= lastind; x++)
        {
            string s = in[x].substr(o, 1);
            char letter = s[0];
            charv.push_back(letter);
            o++;
            if(o > lastind)
                break;
        }
        string s = stringify(charv);
        rotated.push_back(s);
    }


    for (int i = 1; i <= lastind; i++)
    {
        vector<char> charv;
        int o = 0;

        for (int x = i; x <= lastind; x++)
        {
            string s = in[x].substr(o, 1);
            char letter = s[0];
            charv.push_back(letter);
            o++;
            if(o > lastind)
                break;
        }
        string s = stringify(charv);
        rotated.push_back(s);
    }
    return rotated;
}


void PrintGrid(vector<string> grid, int lastIndex)

string stringify(vector<char> v)
{
    int end = v.size();
    char str[end + 1];

    for (int i = 0; i < v.size(); i++)
    {
        str[i] = v[i];
    }
    str[end] = '\0';
    string outstring = string(str);

    return outstring;
}
